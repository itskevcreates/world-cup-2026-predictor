# FIFA World Cup Prediction Platform

An AI-powered platform that predicts FIFA World Cup outcomes — match winners,
scorelines, group standings, knockout qualification, and tournament winners —
using machine learning and Monte Carlo simulation.

## Tech Stack
- **Backend:** Python, FastAPI, PostgreSQL
- **Machine Learning:** Pandas, NumPy, scikit-learn, XGBoost, LightGBM, CatBoost
- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Deployment:** Docker, GitHub Actions, Vercel, Railway
